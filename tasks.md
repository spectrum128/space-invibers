# Space Invaders MVP Tasks

1. **Scaffold project** (30 min)
- Files: `index.html`, `style.css`, `src/main.js`
- Canvas setup and resize handling

2. **Game loop + state** (45 min)
- `requestAnimationFrame`, `deltaTime`
- States: `start`, `playing`, `gameOver`, `win`

3. **Player + controls** (60 min)
- Move left/right with bounds
- Shoot with cooldown (`Space`)

4. **Alien grid + movement** (90 min)
- Spawn formation
- March side-to-side, drop on edge

5. **Projectiles + collisions** (90 min)
- Player bullets hit aliens
- Alien bullets hit player
- Remove dead/inactive entities

6. **Win/lose + lives + score** (60 min)
- Lives decrement and end game
- Score per alien
- Win when all aliens dead

7. **HUD + overlays** (45 min)
- Draw score/lives
- Start and restart prompts

8. **Polish + bug pass** (60 min)
- Basic sounds or placeholder effects
- Tune speeds/cooldowns
- Quick mobile/desktop sanity check

**Estimated total:** ~8 hours
